# Changelog

All notable changes to the SQL Explorer extension will be documented in this file.

## [0.0.1] - 2026-02-03

### Added
- Initial release
- Drag and drop support for CSV, Parquet, JSON, and SQLite files
- SQL editor with Monaco Editor integration
- SQL syntax highlighting
- Dynamic autocomplete for table and column names
- Virtual scrolling results grid for large datasets
- Schema explorer sidebar showing tables and columns
- Export results as CSV or Parquet
- Support for CTEs and complex JOIN queries
- Light and dark theme support
- Keyboard shortcut (Ctrl+Enter) for query execution
